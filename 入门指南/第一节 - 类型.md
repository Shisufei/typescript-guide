# 第一节 - 类型

## 基础

类型系统是 TypeScript 的核心, 在这一节, 我们会按照由简单到不那么简单的顺序学习相关语言特性.

### 基本类型

- 四个原始值类型 `string`, `number`, `boolean`, `void` (对应 `undefined` 或 `null`).
- 任意类型 `any`.

### 声明变量

下述三种方式都能让 TypeScript 的类型系统 (以下简称类型系统) 知道变量 `foo` 的类型为 `string`.

```typescript
var foo: string;
var foo: string = '...';
var foo = '...';
```
